# Hi, I'm Nolhan 👋
### Coder from Brittany 🌧️⛵ | Passionate about Self-Hosting, Bots & Development

Welcome to my GitHub profile! I love building tools to make self-hosting easier, automating workflows, creating bots, and managing media servers.

## 👨‍💻 About me

- 🎓 2nd-year student in **Networks & Telecommunications (BUT)** at IUT Lannion.
- 🛡️ Also pursuing a **Cybersecurity Preparatory Course (CUPGE)** at ENSSAT.
- 🔭 I develop and maintain **[CineVault-Lite](https://github.com/NoNoBzH22/CineVault-Lite)**, an admin interface acting as a bridge between JDownloader 2, SpotDL, and Plex Media Server.
- ⚙️ A huge fan of self-hosting, my personal infrastructure runs on **ZimaOS** (which I highly recommend for self-hosting). I manage all my daily services and containers using **Docker** and **Portainer** (including my **Plex** server).
- 🤖 I also have solid experience in creating, developing, and managing **Discord Bots**.
- 🌱 I'm constantly exploring best practices around **Python**, **JavaScript**, and **Node.js**.
- 📫 How to reach me: [LinkedIn](https://www.linkedin.com/in/nolhan-trehel)
